# Foodidy – Food Delivery Web Application

## Overview

Foodidy is a full-stack food delivery web application that enables users to browse food items, manage orders, and complete secure transactions online. The platform is designed to provide a seamless and responsive user experience across devices.

## Features

* User authentication (Sign up, Login, Logout)
* Product browsing and detailed product views
* Search functionality for food items
* Category-based filtering and sorting
* Wishlist management
* Shopping cart functionality
* Secure payment integration using Razorpay
* Address management for delivery
* Responsive user interface

## Tech Stack

**Frontend:** React.js
**Backend:** Node.js
**Payment Gateway:** Razorpay

## Project Structure

Foodidy
├── frontend        # React application
├── backend         # Mock backend with APIs
├── components      # Reusable UI components
├── pages           # Application pages
├── utils           # Helper functions


## Installation and Setup

1. Navigate to the project directory:

   cd food-delivery-website

2. Install dependencies:

   npm install

3. Start the development server:

   npm start

## Backend Details

The project includes a mock backend that provides API endpoints for:

* Product data handling
* User authentication
* Cart and wishlist operations

This setup allows for efficient frontend development and testing.
